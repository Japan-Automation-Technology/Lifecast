## 1. 直接的な類似サービス

### 結論: 「ショート動画 x クラウドファンディング」を一体化した専用プラットフォームは、海外に存在しない

調査の結果、ショート動画フォーマットとクラウドファンディング機能を統合した専用プラットフォームは、2026年2月時点で海外に確認されなかった。これは、市場に明確な空白(ホワイトスペース)が存在することを意味する。

ただし、以下のように**部分的に近い機能を持つサービス**は複数存在する。

| サービス | 動画形式 | 資金調達形式 | 類似度 | 信頼度 |
|---------|---------|------------|-------|--------|
| Seed&Spark | ピッチ動画(長尺) | リワード型クラファン | 中 | A |
| Clapper | ショート動画 | 投げ銭+EC | 中 | A |
| TikTok Shop | ショート動画 | EC(商品販売) | 低〜中 | A |
| Patreon | 動画コンテンツ | サブスク支援 | 低 | A |

#### Seed&Spark (seedandspark.com)
- **概要**: 映画・映像クリエイター専用のクラウドファンディング+ストリーミングプラットフォーム
- **成功率**: 82%(業界平均の2倍)
- **特徴**: ピッチ動画でプロジェクトを紹介し、資金を調達 → 完成作品をストリーミング配信
- **学び**: クリエイター教育・サポートを徹底(専属クラウドファンディングスペシャリストが全プロジェクトにフィードバック)することで高い成功率を維持
- **自社との差異**: 動画はピッチ用(長尺)であり、ショート動画フォーマットではない
- 信頼度: A / 2024年データ

#### Clapper
- **概要**: TikTokの成熟層向け代替アプリ(ショート動画プラットフォーム)
- **MAU**: 1,000万人超(2025年1月時点)
- **収益モデル**: 投げ銭(チップ) + ライブ配信ギフト + Clapper Shop(EC)
- **手数料**: ライブ配信物販5%、サブスクリプション/投げ銭15%
- **特徴**: 広告なしモデル。視聴者の投げ銭がクリエイター収益の中心
- **自社との差異**: クラウドファンディング(プロジェクト支援・目標金額設定)の概念がない。あくまで「チップ型」
- 信頼度: A / 2025年1月データ

## 3. 投げ銭/ライブコマースの海外事例

### 3-1. TikTok Shop

| 指標 | 数値 | 出典時期 |
|-----|------|---------|
| グローバルGMV | $66B(2025年) | 2025年 |
| 米国GMV | $15.82B(2025年) | 2025年 |

| クリエイター総収益 | $4.1B(2024年)→$5.7B予測(2025年) | 2025年 |

**TikTok Live Gifts/投げ銭**:
- フォロワーがライブ中にバーチャルギフトを送信 → Diamondsに変換 → 現金化
- クリエイター取り分: 約50%
- 月間収益事例: $20〜$300/セッション、最大$34,000/月
- Creator Rewards Program: 1,000再生あたり$0.40〜$1.00(長尺動画は最大$6/1K再生)

**成功要因**:
- アルゴリズム主導のコンテンツ発見(フォロワー数に依存しない)
- 動画視聴 → 商品購入の「シームレスな導線」
- 米国ではショート動画(プリレコーディング)が売上の2/3を占める(ライブ依存ではない)
- 信頼度: A / eMarketer, CapitalOneShopping

## 5. 成功/失敗要因の分析

### 5-1. 「動画 x 支援/決済」で成功しているモデルの共通点

| 成功要因 | 該当サービス | 詳細 |
|---------|------------|------|
| **シームレスな導線** | TikTok Shop, 抖音 | 動画視聴→購入/支援が1タップ。フリクションの最小化 |
| **アルゴリズム主導の発見** | TikTok, 抖音, 快手 | フォロワー数に依存せず、コンテンツ品質で拡散 |
| **クリエイター教育/サポート** | Seed&Spark, Kickstarter | 成功率に直結。Seed&Sparkの82%は教育投資の成果 |
| **透明性と信頼** | Patreon, Ko-fi | 支援金の使途を可視化。手数料の明確化 |
| **低い参入障壁** | Buy Me a Coffee, Ko-fi | 「コーヒー1杯」のメタファー。心理的ハードルの低減 |
| **複数収益源** | Twitch, TikTok | サブスク+投げ銭+広告+EC。単一依存の回避 |
| **エモーショナルな接続** | YouTube Super Chat, Twitch | リアルタイムの双方向コミュニケーション |
| **ネットワーク効果** | 抖音, TikTok | ユーザー増→クリエイター増→コンテンツ増の好循環 |

### 5-2. 失敗/課題要因

| 失敗要因 | 事例 | 詳細 |
|---------|------|------|
| **高い手数料への不満** | GoFundMe(デフォルトチップ10〜19%) | 寄付者が手数料に不信感を持ち離脱 |
| **単一セクター依存** | Koregraf(不動産特化で破綻) | 市場変動に対する脆弱性 |
| **透明性の欠如** | 複数プラットフォーム | 支援金の使途が不明→信頼喪失→支援減少 |
| **寄付疲れ(Donation Fatigue)** | 汎用プラットフォーム全般 | 過度な支援要請がユーザーの疲弊を招く |
| **品質管理の困難** | TikTok Shop(偽造品問題) | 商品品質の担保がプラットフォームの信頼性に直結 |
| **クリエイター収益の低さ** | 旧TikTok Creator Fund($20〜40/100万再生) | 収益が低すぎてクリエイター離れ→改善(Rewards Program) |
| **互恵性の限界** | Twitch(ギフトサブの研究) | ピア間のギフト交換はストリーマーへの寄付に繋がらない |

### 5-3. 各サービスの成否サマリー

| サービス | 成否 | 主要因 |
|---------|------|--------|
| TikTok Shop | 成功 | 動画→EC導線の完成度。アルゴリズム。GMV $66B |
| 抖音(Douyin) | 大成功 | スーパーアプリ化。国内市場の71%が利用 |
| YouTube Super Chat | 成功 | 既存の巨大ユーザーベース。70%のクリエイター取り分 |
| Twitch | 成功(やや停滞) | ライブ×投げ銭の先駆者。ただし50%手数料に不満 |
| Patreon | 成功 | サブスクモデルの確立。年間$2B超のクリエイター収益 |
| Seed&Spark | ニッチ成功 | 映像クリエイター特化。82%の高成功率 |

| Kickstarter | 成功 | 動画付きで成功率1.67倍。$706M調達(2024年) |
| Ko-fi | 成功 | 手数料0%モデルで差別化 |

## 6. 日本市場への適用可能性と学び

### 6-1. 日本市場の特性

- **既存クラファン市場**: CAMPFIRE(82B円超)、Makuake(年間13B円)、READYFORが三強
- **市場規模**: 2024年推定137億円 → 2030年360億円予測
- **特徴**: 日本のクラファンは「予約購入」(Makuake)の色彩が強く、欧米型の「プロジェクト支援」とは文化的に異なる
- **投げ銭文化**: 日本ではVTuber・ライブ配信(YouTube、SHOWROOM、17LIVE等)での投げ銭が定着済み

### 6-2. 自社プラットフォームへの学び

#### 必須要素（成功サービスの共通項）

1. **シームレスな動画→支援導線**
   - TikTok Shopの最大の教訓: 動画視聴から支援/購入まで画面遷移を最小化
   - 「2タップ以内」で支援完了が理想

2. **アルゴリズムによるコンテンツ発見**
   - フォロワー数ではなくコンテンツの質で露出が決まる仕組み
   - 小規模クリエイターでも「バズる」可能性がある設計
   - Kickstarter/Indiegogo型の「自分で集客」モデルでは成功率が低い

3. **クリエイター教育・サポート体制**
   - Seed&Sparkの82%成功率は、専任スペシャリストによる個別サポートの成果
   - 単にプラットフォームを提供するだけでは不十分

4. **適切な手数料設計**
   - Ko-fi(0%) < Buy Me a Coffee(5%) < Patreon(5〜12%) < Clapper(15%) < YouTube(30%) < Twitch(50%)
   - 手数料の高さはクリエイター離れの主因になりうる
   - 初期はKo-fi型(0〜5%)で参入障壁を下げ、スケール後に段階的引き上げが現実的

5. **透明性の担保**
   - 支援金の使途可視化はPatreonの成功要因
   - プロジェクトの進捗報告機能は必須

#### 差別化ポイント（空白市場を攻めるための独自性）

1. **「ショート動画ファースト」のクラウドファンディング**
   - 既存サービスにはない組み合わせ。テキスト+長尺動画のピッチではなく、15〜60秒の動画でプロジェクトを訴求
   - TikTok世代の視聴習慣に合致

2. **段階的支援モデル**
   - 投げ銭(即時・少額) + クラファン(目標型・中額) + サブスク(継続・定額)を1つのプラットフォームで統合
   - 既存サービスはいずれか1つに特化しており、統合型は存在しない

3. **コミュニティドリブンな目標設定**
   - 個別プロジェクトだけでなく、コミュニティ全体の目標(例: 地域活性化、社会課題解決)を設定できる仕組み

#### リスク・注意点

1. **品質管理**: TikTok Shopの偽造品問題から学び、プロジェクトの審査・品質管理体制が必要
2. **寄付疲れ対策**: 過度な支援要請はユーザー離れを招く。パーソナライズされた推薦で緩和
3. **規制対応**: 日本の資金決済法、特定商取引法への適合が必須
4. **マネタイズの段階的設計**: 初期はクリエイター獲得を優先し、手数料は後から段階的に導入
5. **ライブ vs ショート動画**: 米国TikTok Shopではショート動画が売上の66%、ライブは30%未満。日本でもショート動画中心の設計が妥当

## 総括

**「ショート動画 x クラウドファンディング」を統合した専用プラットフォームは、2026年2月時点で海外に存在しない。** これは明確なホワイトスペースであり、参入機会がある。

ただし、以下の事実を踏まえた慎重な設計が必要:

1. **動画はクラファン成功の最強ツール**: Kickstarter/Indiegogo共に動画付きプロジェクトの成功率は約1.67倍、調達額は2倍以上
2. **投げ銭/EC型は既に巨大市場**: TikTok Shop($66B)、抖音($375B)が証明済み。ただしこれらは「クラファン」ではない
3. **成功の鍵は「導線のシームレスさ」**: 動画視聴→支援のフリクションを最小化すること
4. **クリエイターサポートが成功率を決める**: プラットフォーム提供だけでは不十分。Seed&Sparkの82%成功率がその証拠
5. **手数料設計が競争力を左右**: 0〜5%で参入し、スケール後に段階的引き上げが最も現実的
6. **日本固有の投げ銭文化(VTuber等)との融合**: 既存の文化的基盤を活かせる可能性が高い
