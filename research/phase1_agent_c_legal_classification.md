# Phase 1: 投げ銭・購入型クラウドファンディングの法的分類調査

> 調査日: 2026-02-14
> 対象: ショート動画 x クラウドファンディング プラットフォーム
> 取引類型: 購入型CF（All or Nothing）/ 投げ銭（都度課金）

---

## 1. 購入型クラウドファンディングの法的分類

### 1.1 法的性質

購入型クラウドファンディングは、支援者がプロジェクト実施者に対して金銭を提供し、その対価としてリターン（商品・サービス）を受け取る取引であり、法的には**売買契約（民法555条）または役務提供契約**として整理される。[信頼度A]

**All or Nothing 方式の法的整理:**

- 目標金額到達を**停止条件**とする売買の予約（条件付契約）と解釈可能
- 目標未達の場合は契約不成立 → 支援金は全額返金
- 目標達成の場合は契約成立 → 実行者にリターン提供義務が発生
- 民法127条（条件付法律行為）の適用を受ける

参考: [GVA法律事務所 - クラウドファンディングの法務](https://gvalaw.jp/blog/a20250411/)、[TMI総合法律事務所](https://www.tmi.gr.jp/content/covid-5.3no.26.pdf) [信頼度B]

### 1.2 特定商取引法の適用

**通信販売への該当性: 該当する** [信頼度A]

購入型クラウドファンディングは、インターネットを通じた売買取引であり、特定商取引法が規制する「通信販売」（法2条2項）に該当する。

**プロジェクト実施者（起案者）の義務:**

| 義務項目 | 根拠条文 | 内容 |
|----------|----------|------|
| 広告の表示義務 | 法11条 | 販売業者名、住所、代表者名、販売価格、送料、支払時期・方法、商品引渡時期、返品条件 |
| 誇大広告の禁止 | 法12条 | 品質・性能・効果について著しく事実に相違する表示の禁止 |
| 前払式通信販売の承諾通知 | 法13条 | 代金受領後遅滞なく商品送付できない場合の通知義務 |

**クーリングオフの適用: 適用なし** [信頼度A]

通信販売にはクーリングオフ制度が**原則として適用されない**（特商法に通信販売のクーリングオフ規定がない）。ただし、返品特約の表示がない場合は、商品到着後8日間は送料消費者負担で返品可能（法15条の3）。

参考: [TOPCOURT - 購入型CFを規制する2つの法律](https://topcourt-law.com/finance/purchase-type_crowdfunding_two_laws) [信頼度B]

### 1.3 景品表示法の適用

プロジェクト実施者は景品表示法上の「事業者」に該当し、以下が規制される: [信頼度A]

- **優良誤認表示の禁止**（法5条1号）: リターンの品質・性能を実際よりも著しく優良と示す表示
- **有利誤認表示の禁止**（法5条2号）: 取引条件を実際よりも著しく有利と示す表示
- 消費者庁から合理的根拠資料の提出を求められた場合（法7条2項）、提出できなければ不当表示とみなされる

### 1.4 消費者契約法の適用

支援者が「消費者」、実施者が「事業者」に該当する場合、消費者契約法の適用を受ける: [信頼度A]

- 不実告知（法4条1項1号）による取消権
- 不当条項の無効（法8条〜10条）

## 2. 投げ銭の法的分類

### 2.1 法的性質 — 贈与か役務対価か

投げ銭の法的性質は、**取引の実態に応じて判断が分かれる**グレーゾーンである。 [信頼度B]

| 観点 | 贈与と評価される場合 | 役務対価と評価される場合 |
|------|----------------------|--------------------------|
| 対価性 | 明確な反対給付なし | 「投げ銭したら○○します」等の約束あり |
| 金額決定 | 支援者の任意 | 定額メニューが存在 |
| 継続性 | 単発・散発的 | 配信活動に対する継続的報酬 |
| 義務性 | 支払義務なし | サービス利用の条件として設定 |

**実務上の通説:** 配信者がコンテンツ配信を反復継続して行い、その活動に関連して投げ銭を受領している場合は、**役務対価（所得）として扱う**のが一般的。純粋な贈与として扱われるのは、個人間の私的な金銭移転に限定される傾向。

参考: [あいち刑事事件総合法律事務所 - 投げ銭は贈与？](https://datsuzei-bengoshi.com/%E6%8A%95%E3%81%92%E9%8A%AD%E3%81%AF%E8%B4%88%E4%B8%8E%EF%BC%9F%E7%A8%8E%E9%87%91%E3%81%AF%EF%BC%9F/)、[ソリマチ - 投げ銭の税金](https://letter.sorimachi.co.jp/taxnews/20230502_01) [信頼度B]

### 2.2 税務上の扱い

#### 2.2.1 所得税（受け取る側）

| 配信者の状況 | 所得区分 | 根拠 |
|-------------|----------|------|
| 個人事業主として開業（事業的規模） | **事業所得** | 所得税法27条 |
| 副業・一時的な活動 | **雑所得** | 所得税法35条 |
| 完全な個人間の私的贈与 | **贈与税の対象** | 相続税法21条の5 |

- 2025年（令和7年）12月施行の税制改正により、基礎控除額が95万円に引き上げ（従来48万円）
- 確定申告義務: 所得金額が95万円を超える場合（2025年分以降）

[信頼度A - 国税庁通達に基づく]

#### 2.2.2 消費税

投げ銭が消費税の課税対象となるかは**対価性の有無**で判断される: [信頼度B]

- **不課税（対価性なし）:** 配信に対する見返りが明確でない自発的な投げ銭 → 国税庁の「対価を得て行われる」（No.6113）の要件を満たさない
- **課税（対価性あり）:** 「投げ銭で○○します」等の明確な反対給付がある場合 → 役務提供の対価として課税対象

参考: [国税庁 No.6113 「対価を得て行われる」の意義](https://www.nta.go.jp/taxes/shiraberu/taxanswer/shohi/6113.htm) [信頼度A]

#### 2.2.3 プラットフォーム手数料と税務処理

| 項目 | 処理 |
|------|------|
| 視聴者が支払った総額 | 配信者の「売上（収入金額）」 |
| PF手数料（30〜50%が相場） | 配信者の「支払手数料」として経費計上 |
| 源泉徴収 | PFが配信者への支払時に源泉徴収する場合あり |
| 支払調書 | PFは税務署に支払調書を提出する義務あり（一定額以上） |

**注意:** 税務署はPFからの支払調書、振込記録、配信履歴等を通じて配信者の収入を把握している。

参考: [小谷野税理士法人 - ライバーの確定申告](https://koyano-cpa.gr.jp/nobiyo-kaikei/column/5917/)、[弥生 - ライバーの確定申告](https://www.yayoi-kk.co.jp/shinkoku/oyakudachi/streamer/) [信頼度B]

### 2.3 資金決済法上の規制（プラットフォーム設計の核心）

投げ銭プラットフォームの最大の法的論点は、**資金決済法上の「為替取引」への該当性**である。 [信頼度A]

#### 2.3.1 為替取引の該当性

視聴者 → PF → 配信者 の資金フローは「顧客から隔地者間で直接現金を輸送せずに資金を移動する仕組み」に該当し、**為替取引（銀行法2条2項）**に該当する可能性が高い。

為替取引を業として行うには**資金移動業の登録**（資金決済法37条）が必要:
- 株式会社であること
- 最低保証金1,000万円
- 送金額に応じた100%以上の資産保全義務
- 犯罪収益移転防止法に基づくKYC義務
- 内部管理体制の整備

#### 2.3.2 回避スキーム（2方式）

**方式A: 前払式支払手段（ポイント/アイテム方式）** [信頼度B]

```
視聴者 →[現金]→ PF（ポイント購入）→[ポイント/アイテム]→ 配信者
                                        ※配信者はポイントを直接現金化不可
```

- 為替取引に該当しない設計が可能
- 資金決済法上の「前払式支払手段発行者」としての届出/登録が必要
  - 自家型: 届出（未使用残高1,000万円超の場合）
  - 第三者型: 登録（より厳格）
- 供託義務: 未使用残高の50%以上
- **設計上の注意:** アイテムが実質的に「換金可能」だと為替取引に再分類されるリスク

**方式B: 収納代行方式** [信頼度B]

```
視聴者 →[支払]→ PF（配信者の代理として受領）→[精算]→ 配信者
```

- PFが配信者の「収納代行」として資金を受領する構成
- 為替取引には該当しないとされてきたが、近年の法改正で規制強化の傾向
- 2025年改正資金決済法により、クロスボーダー収納代行は規制対象に

参考: [TOPCOURT - 投げ銭サービスの法律問題](https://topcourt-law.com/new_business/throwing-money-service)、[松田綜合法律事務所 - 資金決済法の実務](https://jmatsuda-law.com/lp/paymentservices/)、[ベリーベスト法律事務所](https://corporate.vbest.jp/columns/8115/) [信頼度B]

## 3. KYC（本人確認）要件

### 3.1 犯罪収益移転防止法の適用範囲

| PFの事業形態 | 犯収法の適用 | 根拠 |
|-------------|-------------|------|
| 資金移動業者として登録 | **適用あり** | 犯収法2条2項29号 |
| 前払式支払手段発行者 | **原則適用なし** | ただし一定の場合に自主的対応推奨 |
| 購入型CF（売買取引の場の提供のみ） | **原則適用なし** | PFは特定事業者に該当しない |
| 金融商品取引業者（投資型CF） | **適用あり** | 犯収法2条2項22号 |

[信頼度A - 法令に基づく]

### 3.2 クリエイター（配信者/実施者）側のKYC

| 取引額/状況 | 必要なKYCレベル | 根拠・理由 |
|------------|----------------|-----------|
| PFが資金移動業者の場合 | **厳格な本人確認** | 犯収法4条（取引時確認義務） |
| 前払式支払手段方式 | **PF独自基準での本人確認** | 法的義務ではないが実務上必須 |
| 購入型CF（起案者） | **特商法の表示義務による事実上のKYC** | 氏名・住所・連絡先の表示義務 |
| 全般 | **振込先口座の確認** | 銀行口座開設時のKYCに依拠 |

**推奨KYCレベル（実務上）:**
- 身分証明書（運転免許証、マイナンバーカード等）
- 住所確認
- 銀行口座情報

### 3.3 支援者（視聴者）側のKYC

| 取引額/状況 | 必要なKYCレベル |
|------------|----------------|
| PFが資金移動業者の場合 | **取引時確認が必要**（犯収法4条） |
| 少額取引（前払式支払手段） | **法的義務なし**（メールアドレス程度で可） |
| クレジットカード決済 | カード会社のKYCに依拠 |
| 高額取引（目安: 10万円超） | **実務上の本人確認推奨** |

[信頼度B - 法令 + 実務慣行]

### 3.4 2027年施行予定の犯収法改正への対応

2024年6月閣議決定「デジタル社会の実現に向けた重点計画」に基づき、2027年4月1日施行予定: [信頼度A]

- **本人確認手法の原則一本化:** マイナンバーカードの公的個人認証サービス（JPKI）に統一
- **廃止予定:** 本人確認書類の画像送信方式（従来のeKYC「ホ方式」）
- **写真なし本人確認書類の原則廃止**

→ PF設計時にJPKI対応を前提とする必要あり

参考: [TRUSTDOCK - 犯罪収益移転防止法](https://biz.trustdock.io/column/amlcft)、[クラファンチャンネル - 犯収法とCF事業](https://www.crowdfundingchannel.jp/article-40016/) [信頼度A]

## 4. 消費者保護

### 4.1 プロジェクト不履行時の支援者保護

**法的枠組み:** [信頼度A]

| 状況 | 支援者の権利 | 根拠 |
|------|-------------|------|
| リターン未提供 | **債務不履行に基づく損害賠償請求、契約解除・返金請求** | 民法415条、541条 |
| リターンの品質不良 | **契約不適合責任に基づく追完請求、代金減額請求、損害賠償、解除** | 民法562〜564条 |
| 虚偽の説明 | **消費者契約法に基づく取消権** | 消費者契約法4条 |
| All or Nothing 目標未達 | **全額返金**（契約不成立） | 契約条件（停止条件不成就） |

**現実的な問題:**
- 実施者が返金能力を欠く場合、債権回収は困難
- 少額事案では訴訟コストが支援額を上回る
- PFは通常、実施者と支援者間の紛争に介入しない

### 4.2 プラットフォームの免責範囲と責任

**PFの基本的立場:** 「売買取引の場を提供するにすぎない」→ 原則としてリターン提供義務を負わない [信頼度B]

**ただし、以下の場合にPFの責任が問われうる:**

| リスク要因 | PFの潜在的責任 |
|-----------|---------------|
| 明らかな詐欺プロジェクトの放置 | 不法行為責任（民法709条）/共同不法行為（民法719条） |
| 審査義務の懈怠 | PFの過失として損害賠償責任 |
| 資金管理上の問題 | 預かり金の管理責任 |
| 不当な利用規約 | 消費者契約法10条による無効 |

参考: [有森FA法律事務所 - CF未達成トラブル](https://aog-partners.com/nettrouble/cloudfundingtomitasseienzyoutoraburu/)、[国民生活センター - リターンが届かない](https://www.kokusen.go.jp/mimamori/mj_mailmag/mj-shinsen426.html) [信頼度B]

### 4.3 利用規約で定めるべき事項

#### 4.3.1 購入型CF関連

| 必須事項 | 内容 |
|---------|------|
| PFの法的地位 | 「売買契約の当事者ではなく、場の提供者である」旨の明記 |
| 実施者の義務 | リターン提供義務、不履行時の返金義務 |
| 免責条項 | PFはリターンの品質・提供を保証しない旨 |
| All or Nothing の条件 | 目標未達時の返金プロセス、達成時の契約成立時期 |
| 特商法表示義務 | 実施者に対する特商法に基づく表示義務の遵守要求 |
| 紛争解決手続 | PFの調停・仲裁への関与範囲 |
| 知的財産権 | コンテンツの権利帰属、利用許諾範囲 |
| 禁止事項 | 詐欺的プロジェクト、法令違反コンテンツ等 |

#### 4.3.2 投げ銭関連

| 必須事項 | 内容 |
|---------|------|
| 投げ銭の法的性質 | 取引の性質（対価か贈与か）の定義 |
| 手数料率 | PFが控除する手数料の率と計算方法 |
| 精算・出金条件 | 最低出金額、精算サイクル、出金手数料 |
| 返金ポリシー | 購入済ポイント/アイテムの返金可否 |
| アカウント停止 | 規約違反時のアカウント停止・残高処理 |
| 未使用残高の取扱い | 有効期限、失効時の処理 |
| 税務上の注意喚起 | 配信者に対する確定申告義務の告知 |

[信頼度B - 弁護士解説 + 主要PFの利用規約分析に基づく]

参考: [CAMPFIRE利用規約](https://camp-fire.jp/term)、[日本政策金融公庫 - CFの法務](https://www.jfc.go.jp/n/finance/social/tokushuu23.html) [信頼度B]

## 5. 規制対応サマリー（PF事業者として必要な対応）

### 5.1 必須対応事項

```
[優先度: 高]
1. 事業形態の決定
   - 為替取引を回避するスキーム設計（前払式支払手段 or 収納代行）
   - 弁護士との協議による法的整理

2. 資金決済法対応
   - 前払式支払手段発行者の届出/登録
   - 供託義務への対応（未使用残高の50%以上）

3. 特定商取引法対応
   - 実施者向けの特商法表示テンプレート作成
   - 広告審査フローの構築

4. 利用規約・プライバシーポリシーの整備
   - 弁護士によるリーガルレビュー必須

[優先度: 中]
5. KYCフローの構築
   - クリエイター: 身分証 + 住所確認 + 銀行口座
   - 支援者: メールアドレス（少額）/ 身分証（高額）
   - 2027年JPKI一本化への対応準備

6. 税務処理体制
   - 支払調書の発行体制
   - 源泉徴収の要否判断

[優先度: 低（将来対応）]
7. 2027年犯収法改正への対応
   - マイナンバーカードJPKI認証の実装
```

### 5.2 法的リスクマトリクス

| リスク | 発生可能性 | 影響度 | 対策 |
|--------|-----------|--------|------|
| 為替取引該当による無登録営業 | 高 | 致命的 | スキーム設計の法的検証 |
| 特商法違反（実施者） | 中 | 中 | テンプレート・審査フロー |
| リターン不履行による消費者トラブル | 高 | 中 | 利用規約・保証金制度 |
| 税務処理の不備 | 中 | 中 | 税理士との連携体制 |
| PF責任の追及 | 低 | 高 | 免責条項・審査体制 |
