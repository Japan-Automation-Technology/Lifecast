## 2. 各社の特徴・強み・弱み

### 2-1. CAMPFIRE

**概要**: 2011年設立。国内最大級のクラウドファンディングプラットフォーム。代表取締役: 中島 真。累計10万件超のプロジェクト、510万人の会員基盤。

| 項目 | 内容 |
|------|------|
| **強み** | |
| 圧倒的な規模 | 累計10万件超のプロジェクト、510万人会員は国内最大 |
| カテゴリの多様性 | 全33カテゴリ対応。音楽、まちづくり、映画、アニメなどクリエイティブ領域に強い |
| 参入障壁の低さ | 審査が比較的緩く、個人・小規模事業者でも始めやすい |
| エコシステム | CAMPFIRE Community（月額課金）、CAMPFIRE Creation（グッズ）、ふるさと納税など多角展開 |
| **弱み** | |
| 品質管理 | 審査の緩さゆえにプロジェクト品質のばらつきが大きい |
| 成功率 | 約43%と中程度。玉石混交で未達成プロジェクトも多い |
| リターン未提供対応 | Makuakeのような独自返金制度がない |

**「ショート動画 x クラファン」への示唆**: CAMPFIREはYouTube Live連携等の動画機能を既に持ち、クリエイター層のユーザーベースが大きい。エンタメ・音楽カテゴリの成長率173%は動画コンテンツとの親和性を示唆。ただし、ショート動画への最適化は未実装。

---

### 2-2. Makuake

**概要**: 2013年サービス開始。2019年12月に東証マザーズ（現グロース市場）上場。株式会社マクアケが運営。「応援購入」をコンセプトとするプロダクト特化型プラットフォーム。

| 項目 | 内容 |
|------|------|
| **強み** | |
| プロダクト特化 | 家電・ガジェット等のプロダクトカテゴリが全体の約47%を占め、EC的な購買体験を提供 |
| 審査品質 | 3段階審査（一次・実現性・表記）で薬機法・景表法審査まで実施。プロジェクト品質が高い |
| サポーターの購買力 | リピーター率が高く（10回以上購入者16万人超）、過去1年の購入者が金額の75%超 |
| 支援者保護 | 「安心システム利用料」による業界唯一の返金制度 |
| 上場企業の信頼性 | IR情報公開、透明性が高い |
| **弱み** | |
| カテゴリの偏り | プロダクト・物販系に強いが、社会貢献・アート系は弱い |
| 手数料の高さ | 20%（税別）は3社中最高 |
| 審査のハードル | 3段階審査で掲載までに時間がかかる。個人起案者にはハードルが高い |

**「ショート動画 x クラファン」への示唆**: Makuakeは成功PJの9割以上が動画を掲載しており、既に動画の重要性が浸透。ただしYouTubeショート非対応を明言しており、縦型ショート動画への対応は未着手。プロダクト紹介との相性はショート動画と非常に高く、ここが競合余地。

### 2-3. READYFOR

**概要**: 2011年3月29日、東日本大震災直後に日本初のクラウドファンディングとしてサービス開始。READYFOR株式会社運営。代表: 米良はるか。社会貢献・寄付型クラウドファンディングの代名詞的存在。

| 項目 | 内容 |
|------|------|
| **強み** | |
| 圧倒的な成功率 | 約75%は業界トップ。社会貢献プロジェクトの共感性の高さが寄与 |
| 社会貢献特化 | NPO、医療・福祉、大学・研究、動物愛護、神社仏閣等で国内シェアNo.1 |
| 手数料の安さ | シンプルプラン12%（税別）は3社中最安 |
| 高単価支援 | 1支援あたり平均1.9万円と高い。共感による高額寄付が多い |
| 大型事例 | 国立科学博物館9.1億円、日本将棋連盟9.4億円等の実績 |
| **弱み** | |

| カテゴリの偏り | 社会貢献系に特化しており、プロダクト・エンタメ系には弱い |

**「ショート動画 x クラファン」への示唆**: 社会貢献プロジェクトは「ストーリー」で共感を得る構造であり、ショート動画との親和性は潜在的に高い。ただし、READYFORのユーザー層（NPO、大学、医療機関等）はショート動画制作リテラシーが低い可能性があり、ツール側のサポートが重要。

## 4. 「ショート動画 x クラファン」への戦略的示唆

### 4-1. 既存3社の動画対応ギャップ

3社とも以下の共通課題を抱えており、ここが新規参入の最大の機会窓:

1. **ショート動画（縦型・短尺）非対応**: 3社ともYouTube/Vimeo埋め込みが前提。TikTok/Instagram Reels/YouTube Shorts的なネイティブ縦型動画体験がない
2. **動画ファイル直接アップロード非対応**: 全て外部プラットフォーム経由。UX上の摩擦が大きい
3. **動画によるディスカバリー機能なし**: 動画フィードによるプロジェクト発見体験が存在しない
4. **動画コンテンツの分析・最適化なし**: 動画の視聴データとコンバージョン（支援）の連動分析機能がない

### 4-2. 参入戦略上の考慮点

| 検討事項 | 内容 |
|----------|------|
| **差別化ポイント** | ショート動画フィードによるプロジェクト発見体験は3社とも未実装 |
| **手数料設計** | 業界水準は12%〜22%。15%前後が競争力のあるライン |
| **初期カテゴリ** | Makuakeのプロダクト系（動画映え）とCAMPFIREのエンタメ系（動画親和性高）の掛け合わせが有力 |
| **審査方針** | Makuakeの厳格審査は品質を担保するがスケールしにくい。CAMPFIREの緩い審査はスケールするが品質リスクあり。中間を取りつつAI審査で効率化が理想 |
