# User Journeys

## Creator journey (MVP)
1. Complete account and identity requirements.
2. Create project with required fields.
3. Define up to 3 support plans.
4. Publish short progress videos.
5. Respond to comments and maintain progress updates.
6. Reach funding goal or fail all-or-nothing condition.

## Supporter journey (MVP)
1. Watch vertical feed.
2. Tap support from video card.
3. Confirm key project facts on support confirmation card.
4. Select plan and complete payment.
5. Receive supporter badge visibility and support history entry.

## Moderation journey (MVP)
1. Receive trust-weighted reports or detect policy violation.
2. Evaluate evidence and logs.
3. Stop project when conditions match policy.
4. Notify creator and allow 72-hour appeal explanation.
5. Record final decision.
