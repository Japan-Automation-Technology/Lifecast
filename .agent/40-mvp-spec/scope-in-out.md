# Scope In and Out

## In scope (MVP)
- Vertical short-video feed.
- Creator project creation.
- Plan creation up to 3 tiers.
- Support flow with external checkout and Apple Pay support.
- All-or-nothing funding and automatic failure refunds.
- Like/comment/share.
- Account creation and follow.
- Supporter badge display.
- Supporter profile list of supported projects.
- Email/phone verification.
- Creator identity document upload.
- Project stop operation.
- Funding progress and days-left display.
- Basic notifications.
- Core conversion and activity logging.

## Out of scope (MVP)
- Long-form video.
- Live streaming.
- Direct messaging.
- Polling/voting.
- Supporter-only exclusive content.
- Advanced analytics dashboard.
- Early bird and quantity-limited plans.
- Highly tuned recommendation optimization.
