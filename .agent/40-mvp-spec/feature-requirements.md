# Feature Requirements

## Feed and content
- Vertical swipe short-video feed.
- Video card shows days left and funding progress bar.
- Creator profile has two tabs:
  - Video tab
  - Home tab (active project details)

## Project and plans
- Project required fields include:
  - Goal amount
  - End date
  - Minimum plan summary (price + reward summary)
- Plan count capped at 3 in MVP.

## Social and identity
- Follow creators.
- Like and comment on videos.
- Share content externally.
- Supporter badge shown in comment context.
- Supporter profile shows supported projects.

## Funding and payment
- Purchase-style support plans.
- External payment checkout + Apple Pay support.
- Payment success is canonical support conversion signal.
- All-or-nothing funding settlement behavior.
- Auto refund on failed target outcome.

## Trust operations
- Creator ID document upload for project creators.
- Project stop functionality.
- Basic report-driven moderation operations.

## Notifications
- Basic lifecycle notifications (support confirmation, project updates, key status changes).
